## 📦 Getting Started

Clone the repository

```bash
git clone https://github.com/SourikSasmal/pixel.me.git
```

Move into the project

```bash
cd pixel.me
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
